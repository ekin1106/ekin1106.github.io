centos安装chrome的记录

发表于 2018-05-12

centos中安装chrome的时候，可以直接从chrome的主页下载rpm包

https://dl.google.com/linux/direct/google-chrome-stable_current_x86_64.rpm

后续安装时，会出现依赖包缺失，其实可以直接执行

yum install google-chrome-stable_current_x86_64.rpm

另外yum provides */lsb_release用这个命令也可以找到需要安装的依赖包的名字
