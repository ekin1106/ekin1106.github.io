安装需要的软件

Hexo 刚刚部署完成，现在就部署时候的问题做点记录，已备以后还需要

  1. NodeJs

  2. GIT

  3. 安装完成以后的next主题地址(https://github.com/theme-next/hexo-theme-next)

INSTALL

  • NodeJs 和 GIT 安装时候，按照默认一路next直到安装完成。
  • 使用下面命令来查看是否安装完成

HEXO的安装

新建一个目录/HEXO ，进入目录后打入命令 npm install hexo-cli -g

然后输入npm install hexo --save 如果出现error，提示不是一个空文件夹，就删除文件夹里的内容

HEXO的命令

初始化hexo init然后输入npm install

生成网页hexo g

depoly到git hexo d

主题设置

​ 安装next主题前，在themes里自建next目录

​ 输入git clone https://github.com/theme-next/hexo-theme-next themes/next

​ update的方法，进入themes/next 输入git pull

设置启动

修改next目录下_config.yml，找到主题\# Scheme Settings，将#去掉就可以
