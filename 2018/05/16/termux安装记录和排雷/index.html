termux安装记录和排雷

发表于 2018-05-16

首先国际惯例上个图

Termux终于搞定了adb自己连自己，然后就可以用python做的小脚本，自动控制设备了，还带图形匹配，哈哈哈

记录下安装方法和一些简单的设置

安装SSH

安装完termux以后，我没有立即升级，先pkg install openssh ，然后在xshell中生成公钥文件，传输到了手机

因为termux的终端不支持密码认证，所以需要把生成的pub key放在授权登陆里，具体做法

#更换源

直接更换源，方便后面的update，使用中科大的termux源

sed -i 's,http://termux.net,http://mirrors.ustc.edu.cn/termux,' $PREFIX/etc/apt/sources.list

更新完以后重启下termux,然后执行apt update apt upgrade

更改shell样式

在git上找到了一个安装脚本，一键搞定，要是有安装错误，可以先安装ZSH,curl

sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)"
